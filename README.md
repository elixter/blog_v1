# blog_v1
blog_v1

Framework : Echo<br>
API : CKEditor4, Fontawesome<br>
Database : MYSQL<br>
Backend: Golang<br>
Frontend: HTML, CSS, Javascript<br>

GO연습겸 백엔드 연습용으로 만드는 블로그 프로젝트<br>
추후 여러 기능(라이브스트리밍, 라이브채팅, 프로젝트소개 등)이 추가된 종합 서버로 만들예정.<br>

# 사이트 주요기능
- 게시글 작성, 삭제, 수정
- 관리자 페이지(미구현)
- 메일 발송(미구현)
- 영상 스트리밍(미구현)
- 라이브 채팅(미구현)
