# Platform project
- 기존 Go기반 블로그 서버에서 Java Spring boot와 typescript react로 전환 작업중

## 기술스택
- Backend : Java11 Spring boot
- Frontend : Typescript, React
- Database : MySQL

## 기능
- [ ] 로그
- [ ] SNS 통합 피드
